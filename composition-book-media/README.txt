Composition Book Media website

Files included:
- index.html (front cover landing page)
- inside-front-cover.html
- projects.html
- capabilities.html
- inside-back-cover.html
- contact.html
- assets/style.css

How to use on GitHub Pages:
1. Upload every file exactly as-is.
2. Keep the assets folder in the same top-level folder as the HTML files.
3. Edit the placeholder contact info in contact.html.
4. Replace placeholder project text in projects.html with your real projects.

This version uses:
- HTML for structure
- CSS for all styling
- No JavaScript required

That means it should be simple, fast, and easy to host on GitHub Pages.
